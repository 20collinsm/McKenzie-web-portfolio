Usability Study
Yale Art Department

  • Method

    • Target Audience
      • Yale art students and staff
    • Representative Tasks
      • Task 1: find the summer programs
      • Task 2: find a beginnners art course
      • Task 3: find out who to contact to enroll in the course

  • Results

    • Task 1
      • We scrolled around the hope page and the courses page for a while before realizing that there was a tab for summer courses in the directory.
    • Task 2
      • Summer programs were split into 2 categories, Norfolk and New Haven. Norfolk was a dead end and New Haven had course information, but none on the beginner level. We then went back to the courses tab and still could not find it. We never completed this task.
    • Task 3
      • We never attempted to complete this task.

  • Discussion

    • This is the worst web page design I have ever used. If it was not for a class assignment, I would have left the second I had opened the home page. It is impossible to find anything and there is not standard design, leaving the user to guess and check in order to find content. The website looks artistic, which is obviously what they were going for, but the Yale art department is in desperate need of organization skills.

Usability Study
Bordentown Regional School District

  • Method

    • Target Audience
      • Bordentown students, staff, and parents
    • Representative Tasks
      • Task 1: find the middle school math department
      • Task 2: find someone who teaches algebra 1
      • Task 3: find their contact information

  • Results

    • Task 1
      • There was no tab for the math department, so I had to search math in the staff directory to find the math teachers' websites.
    • Task 2
      • There were not any search results for "algebra 1" so I had to find the teachers by trial and error.
    • Task 3
      • The teacher websites had contact information that was easy to access.

  • Discussion

    • For the most part, the website was organized and easy to use. However, I would recommend making a directory for the different departments within the school. This would have helped me save time when I was looking for the middle school math department. Teachers should also make the specific class that they teach more prominent on their web page.

Usability Study
Technology Student Association

  • Method

    • Target Audience
      • TSA members and advisors
    • Representative Tasks
      • Task 1: find the national conference tab
      • Task 2: find information about the 2014 conference
      • Task 3: find their contact information

  • Results

    • Task 1
      • We easily found the tab in the middle of the home page in bold letters.
    • Task 2
      • We scrolled around on the page for a minute before finding a link to the 2014 conference information.
    • Task 3
      • The contact information was directly displayed when the page loaded.

  • Discussion

    • This website was very easy to use and allowed me to find everything that I was looking for within minutes.
